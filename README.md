# processamento-imagens
Atividade de processamento de imagem - código utilizado no google colab
-
Abra o arquivo "Atividade.ipynb" e clique em "open in colab"
-
Após a abertura do notebook no google colab, clique no botão "Executar Tudo", os códigos serão executados e a imagem será editada conforme as atividades pedem
